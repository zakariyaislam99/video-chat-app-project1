# VideoChat - Приложение для видеозвонков и чата

Приложение для видеозвонков и обмена сообщениями, похожее на Skype.

## Возможности

- ✅ Регистрация и авторизация пользователей
- ✅ Обмен текстовыми сообщениями в реальном времени
- ✅ **Групповые беседы** - создавайте группы и общайтесь с несколькими людьми одновременно
- ✅ Видеозвонки через WebRTC (один на один)
- ✅ Аудиозвонки
- ✅ Список контактов с фильтрацией
- ✅ Поиск пользователей
- ✅ Управление видео/аудио во время звонка
- ✅ **PWA (Progressive Web App)** - устанавливается как приложение на мобильные устройства и компьютеры
- ✅ Офлайн режим с локальным хранилищем
- ✅ **Редактирование профиля** - изменение аватара, имени, email и пароля
- ✅ **Админская панель** - управление пользователями и правами доступа
- ✅ **Совещания** - полноэкранный режим для совещаний

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер:
```bash
npm start
```

Или в режиме разработки:
```bash
npm run dev
```

3. Откройте `index.html` в браузере или используйте локальный сервер:
```bash
npx http-server .
```

## Использование

1. Зарегистрируйте новый аккаунт
2. Войдите в систему
3. Выберите контакт из списка или создайте группу (кнопка ➕)
4. Начните чат или видеозвонок
5. Для установки как приложение: в браузере выберите "Установить приложение" или "Add to Home Screen"

## Админ панель

Для входа в админ панель:
1. Зарегистрируйтесь с email `admin@admin.com` или
2. Нажмите кнопку ⚙️ в шапке и введите:
   - Логин: `admin`
   - Пароль: `admin`

В админ панели вы можете:
- Просматривать всех пользователей
- Выдавать права доступа к совещаниям

## Технологии

- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Backend**: Node.js, Express
- **WebSocket**: ws (для реального времени)
- **WebRTC**: для видеозвонков
- **Аутентификация**: JWT

## Примечания

- Приложение работает в офлайн режиме с использованием localStorage, если сервер недоступен
- Для видеозвонков требуется HTTPS (или localhost)
- В продакшене необходимо использовать базу данных вместо массива в памяти
- PWA работает только через HTTPS или localhost

## Деплой в интернет

Подробные инструкции по деплою на различные платформы смотрите в файле [DEPLOY.md](DEPLOY.md)

Быстрый деплой на Railway:
1. Зарегистрируйтесь на [Railway.app](https://railway.app)
2. Создайте новый проект из GitHub репозитория
3. Railway автоматически определит Node.js и задеплоит
4. Получите URL вашего приложения
5. Обновите WebSocket URL в `app.js` на ваш домен

## Создание иконок для PWA

Откройте `create-icons.html` в браузере и скачайте иконки, или создайте их вручную размером 192x192 и 512x512 пикселей.
