# Быстрый старт VideoChat

## Локальный запуск

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите сервер:**
   ```bash
   npm start
   ```

3. **Откройте приложение:**
   - Откройте `index.html` в браузере
   - Или используйте локальный сервер: `npx http-server . -p 8080`
   - Откройте `http://localhost:8080`

4. **Создайте иконки для PWA (опционально):**
   - Откройте `create-icons.html` в браузере
   - Нажмите кнопки для создания иконок
   - Сохраните файлы в корневую папку проекта

## Тестирование

1. Откройте приложение в двух разных браузерах (или вкладках в режиме инкогнито)
2. Зарегистрируйте двух разных пользователей
3. Начните чат между ними
4. Создайте группу и добавьте участников
5. Попробуйте видеозвонок (требуется разрешение на камеру/микрофон)

## Деплой в интернет

Смотрите подробные инструкции в [DEPLOY.md](DEPLOY.md)

**Самый простой способ - Railway:**
1. Зарегистрируйтесь на railway.app
2. Создайте проект из GitHub
3. Готово! Railway автоматически задеплоит

## Важно

- Для видеозвонков нужен HTTPS (или localhost)
- PWA работает только через HTTPS
- В продакшене используйте базу данных вместо массива в памяти
